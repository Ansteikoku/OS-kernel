# PolyglotOS (bubblesOS & PysOS)

C言語で書かれた、PHPとPythonのシミュレーションエンジンを搭載したマルチ言語OSプロトタイプです。

## 開発環境の構築

ビルドには i686-elf クロスコンパイラが必要です。

```bash
brew install i686-elf-gcc i686-elf-binutils nasm xorriso qemu  # Homebrewを使用する場合
```

## UTM (macOS) での実行方法

UTMで起動する場合、以下の設定を行わないとキーボードが認識されません。

1. **新規作成**: 「エミュレート」→「その他」を選択。
2. **ISO選択**: `polyglot.iso` を選択し、**「UEFIブートを使用」のチェックを外して**作成。
3. **システム設定**:
   - **マシン (Machine)**: `Standard PC (i440FX + PIIX, 1996) (pc)` を選択。
4. **ドライブ設定**:
   - **インターフェース**: `IDE` を選択。
5. **入力設定**:
   - **USBサポート**: **OFF** にする。

## キーボードが動作しない場合

UTMの設定が難しい場合や、キーボード入力を受け付けない場合は、**QEMU** での実行を推奨します。QEMUは標準的なPS/2コントローラをエミュレートするため、最も安定して動作します。

```bash
make run
```

## 使い方

起動後のメニューでOSを選択してください。
- `help`: コマンド一覧を表示
- `php`: PHPエンジンのデモを開始
- `python`: Pythonエンジンのデモを開始
- `cls`: 画面をクリア
